still learning Svelte stuff...

## Features

- classically layout with, Header, Footer, Sidebar & Content
- Sidebar is hiding on small screens
- responsive via matchMedia object (`+layout.svelte`)
- dark & light Theme (incl Code-Highlighting)
- ...
